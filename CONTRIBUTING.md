# Contributing

## Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run checks:
   ```bash
   npm run lint
   npm test
   npm run build
   ```
3. Run the demo app:
   ```bash
   npm run demo:dev
   ```

## Pull Request Guidelines

- keep changes scoped and well-tested
- add or update docs when behavior changes
- prefer simple abstractions over framework-heavy patterns
- avoid duplicating generic logic available in shared packages

## Commit Style

Use clear commit messages that explain intent and impact.
